# Mental Health Assessment Platform

AI-powered mental health assessment platform built with React 19 and TypeScript. Features an interactive survey, real-time ML predictions via Gradio, and comprehensive mental health resources.

## Features

- **AI-Powered Assessment**: 8-question survey with real-time analysis using Gradio ML model
- **Research Hub**: Curated articles, journals, and growth resources with verified links
- **Statistics & Insights**: WHO and Our World in Data visualizations with source citations
- **Resource Directory**: Crisis helplines, support organizations, and self-care tools
- **Assessment History**: View and manage past assessment results stored locally
- **Responsive Design**: Fully responsive across mobile, tablet, and desktop

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Tech Stack

- **React 19** + TypeScript
- **React Router** for navigation
- **Tailwind CSS** for styling
- **GSAP** for animations
- **Gradio Client** for ML model integration
- **Recharts** for data visualization

## ML Model

The platform uses the Gradio Space: `ourafla/Mental-Health-Detection-Model-Interface`

Model Card: [Hugging Face](https://huggingface.co/ourafla/mental-health-bert-finetuned)

## Deployment

Deploy to Vercel:
1. Push code to GitHub
2. Import repository in Vercel
3. Deploy (auto-detects React settings)

The `vercel.json` file is already configured.
